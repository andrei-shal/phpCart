## 🚀 Стек технологий
* **Backend:** PHP 8.2 (ООП, MVC-like подход, паттерн Singleton для БД)
* **Frontend:** JavaScript (ES6 Modules, Fetch API), HTML5, CSS3 (Bootstrap 5.3)
* **База данных:** MySQL

---

## 📋 Реализованные требования (По ТЗ)
1. **Динамическая корзина:** Изменение количества товара происходит без перезагрузки страницы (через API). Стоимость позиции и общая сумма пересчитываются на лету.
2. **Автоудаление:** Если количество товара в корзине уменьшается до `0`, товар автоматически удаляется из сессии.
3. **Хранение данных:** Корзина реализована на стороне бэкенда через сессии (`$_SESSION['cart']`). Данные о товарах хранятся в MySQL.
4. **Форматирование цен:** Цены хранятся в БД в копейках (целое число), а на сайте выводятся через метод форматирования с разделением тысяч и подстановкой валюты.
5. **Валидация полей (Бэкенд):**
    * `Имя` — только буквы (латиница, кириллица, включая "Ё/ё") и пробелы.
    * `Email` — проверка по строгой маске email.
6. **Оформление заказа:** Данные формы отправляются встроенным методом PHP `mail()`. В письме дублируются контакты клиента и состав заказа.
7. **Архитектура:** Единственная точка входа для всех запросов корзины — `api.php`. Подключение к БД инициализируется один раз через Singleton-класс `Database`.
8. **Адаптивность:** Верстка полностью адаптирована под мобильные устройства (использована сетка Bootstrap 5).
